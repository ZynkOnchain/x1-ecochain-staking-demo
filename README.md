## Features
- ERC20 Token (EcoTokenV2)
- Faucet token distribution
- Multi-pool staking system
- Flexible staking (2.5% APY)
- 30 Day Pool (3% APY)
- 90 Day Pool (10% APY)
- 180 Day Pool (25% APY)
- 365 Day Pool (80% APY)
- Early withdrawal penalties
